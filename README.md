# mnn-fact-finder

## Setup Instructions
1. Clone repository
2. Get an API key from the [United States Census Bureau's Developer page](http://api.census.gov/data/key_signup.html)
3. Create a file `settings.json` in the public directory with
 ```javascript
{"KEY" = "<your-key>"};
 ```
4. `npm install`
5. `npm start`
6. Navigate to `localhost:3000`
